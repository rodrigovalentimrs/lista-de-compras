Lista de Compras Inteligente

Aplicação web desenvolvida com HTML, CSS e JavaScript Vanilla para gerenciamento de listas de compras.

Sobre o projeto

O sistema permite adicionar, editar, remover e selecionar produtos dinamicamente, realizando cálculos automáticos de valores totais da compra.

O projeto foi criado com foco em consolidar fundamentos de desenvolvimento front-end e arquitetura de aplicações JavaScript.
